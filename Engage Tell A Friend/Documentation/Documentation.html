Engage TellAFriend Module

* Important *

When creating a project using this template, name your project simply "ModuleName", not "EngageModuleName".

Due to limitations in the templating capabilities of Visual Studio, you will need to make some minor changes before you can build this module.

  • BEFORE checking the module into TFS, rename the project folder to EngageTellAFriend.
  • Edit the Project properties by double-clicking on the Properties folder in Solution Explorer
  • On the Application tab:
    • Make sure the Assembly name textbox is set to "EngageTellAFriend".
    • Make sure the Default namespace is set to "Engage.Dnn.TellAFriend".
  • On the Web tab:
    • Under 'Start Action', make sure to set your Start URL to the correct site.
    • Under 'Servers', make sure to set up your IIS Project Url.

Also, be sure to follow the Module Development Standards.
